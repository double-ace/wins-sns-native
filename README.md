## ローカル開発環境の構築

### （まだExpo導入していない場合）Expo CLI のインストール

<a href="https://expo.dev/" target="blank">Expo公式サイト</a>

```bash
npm install -g expo-cli
```

バージョンが表示されれば OK

```bash
expo --version
4.12.0 # 例
```

1. リポジトリのクローン

作成したい場所にカレントディレクトリを移動させて下記を実行

```bash
git clone https://github.com/naoya9411/mahjong-sns.git
```


2. 依存関係のインストール

```bash
npm install
```


3. expo 起動

```bash
npm run start
```
