## ローカル開発環境の構築

1. リポジトリのクローン

作成したい場所にカレントディレクトリを移動させて下記を実行

```bash
git clone https://github.com/naoya9411/mahjong-sns.git
```

2. Expo CLI のインストール

[Expo 公式サイト](https://expo.dev/)

```bash
npm install -g expo-cli
```

バージョンが表示されれば OK

```bash
expo --version
4.12.0 # 例
```

3. 依存関係のインストール

```bash
npm install
```

4. expo 起動

```bash
npm run start
```
